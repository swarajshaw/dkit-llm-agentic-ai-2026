# DKIT LLM Agentic AI 2026

This repository is my one-year course operating system for:
- learning notes
- weekly assignments
- hands-on labs
- production-grade AI projects

## Structure
- `notes/`: lecture notes and experiments
- `assignments/`: weekly tasks
- `labs/`: focused mini-builds
- `projects/`: shippable systems and capstone
- `adr/`: architecture decisions
- `infra/`: Docker, Terraform, and Kubernetes assets

## End Goal
Build and deploy an enterprise-grade AI application with:
- RAG
- agents and workflows
- evals in CI
- security controls
- observability and cost controls
